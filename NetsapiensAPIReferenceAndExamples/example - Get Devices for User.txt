In this example, we are getting all logical devices for a specific user (FXS 4 / 1004)


Request:

```
import netsapiensApi from '@api/netsapiens-api';

netsapiensApi.auth('nss_aXJTse8mo02cfiihvKjFfp4ohRg1BuiG34CKMY4uBGXcIiFTa362a929');
netsapiensApi.getDevices({domain: 'exampledomain.com', user: '1004'})
  .then(({ data }) => console.log(data))
  .catch(err => console.error(err));
```
